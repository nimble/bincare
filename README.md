# BinCare.ca

Professional bin cleaning service website with waitlist signup.

## Features

- **Landing Page** - Matches bincare.ca's professional layout and style
- **Waitlist Form** - Opt-in form for customer interest (Full Name, Email, Phone Number)
- **NiceJob Reviews Integration** - Display customer reviews and ratings
- **Interactive Chatbot** - Auto-answer common questions about the service
- **Process Section** - Step-by-step explanation of the cleaning process
- **Responsive Design** - Mobile and desktop optimized
- **QR Code Ready** - Perfect for pamphlet distribution

## Getting Started

Simply open `landingpage.html` in a web browser to view the website.

## Structure

- `landingpage.html` - Main landing page
- `styles.css` - Styling and design
- `script.js` - Form handling and interactions
- `chatbot.js` - Chatbot functionality for auto-answering questions

## NiceJob Integration

To enable the NiceJob reviews widget:

1. Sign up for a NiceJob account at https://get.nicejob.com/
2. Get your business ID from your NiceJob dashboard
3. In `landingpage.html`, replace `YOUR_NICEJOB_ID` with your actual NiceJob business ID (line 45)

Alternatively, you can use NiceJob's embed script method if provided by their platform.

## Chatbot

The chatbot automatically answers questions about:
- Service details and process
- Pricing and service plans
- Launch date (March 2025)
- Contact information
- Location/service area
- Benefits of bin cleaning
